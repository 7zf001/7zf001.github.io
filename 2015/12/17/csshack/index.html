css hack

发表于 2015-12-17 | 分类于 css |
  1. 加上一个下划线“_”，IE 6 照样可以识别该属性，而且只有IE 6可以识别。
  2. 特定的css属性值后缀“\9”，从而做到只有IE8及以下浏览器能够识别该属性
  3. IE 7也支持在选择器前添加+html和:first-child+html ，让当前选择器成为*+html的后辈选择器，只有IE
  4. 可以识别此类样式选择器
  5. 只有IE8支持嵌套如下@media类型查询语句：@media \0screen
  6. background-color:red\0;IE8和IE9都支持；
  7. background-color:blue\9\0; 仅IE9支持；
